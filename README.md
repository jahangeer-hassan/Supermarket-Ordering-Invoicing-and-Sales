# 🛒 Supermarket Ordering, Invoicing, and Sales Analysis

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on multiple supermarket business datasets using Python and Google Colab.

The analysis was conducted on three different datasets:

* Invoices Dataset
* OrderLeads Dataset
* SalesTeam Dataset

The main objective of this project was to analyze:

* Customer ordering behavior
* Sales lead conversion performance
* Meal pricing trends
* Company order activity
* Sales team management
* Business growth patterns

Through data cleaning, preprocessing, statistical analysis, and visualization techniques, meaningful business insights were extracted from raw datasets.

## 1️⃣ Invoices Dataset

The Invoices dataset contains:

### Analysis Performed

* Meal price distribution
* Meal type analysis

* Top company activity
